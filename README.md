# Summary

Pandora's Hobby is a follower overhaul mod for Battle Brothers: Reforged that completely replaces the default follower system, adds new champions along with dozens of new drops, recipes, etc.

# Nexus Mod Page
Main mod page can be found on Nexus Mods:
https://www.nexusmods.com/battlebrothers/mods/992

# Important: 
The Follower images have been split off into a separate mod pack, that must be downloaded from the Nexus mod page.
